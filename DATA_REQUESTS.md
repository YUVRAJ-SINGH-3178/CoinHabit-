# Data Requests

CoinHabit supports user requests for data export and account deletion.

## How to Request

1. Open the app
2. Go to **Profile > Settings**
3. Choose **Request My Data Export** or **Delete Account & Data Request**

This opens a pre-filled GitHub issue for the request.

## Required Information

- Email address associated with the account
- Request type (export or deletion)
- Optional details

## Response Time

Requests are processed as quickly as possible, usually within a few business days.
