# Privacy Policy

CoinHabit is designed to respect user privacy.

## Data We Collect

- Account identifiers (email, user ID)
- Savings goals and progress
- App usage metrics (for feature performance)

## How Data Is Used

- To deliver app features and personalization
- To improve reliability and performance

## Data Sharing

We do not sell user data. Data is shared only with service providers needed to operate the app (e.g., Supabase, notification services).

## Data Retention

Data is retained as long as the account is active. Users can request deletion at any time.
