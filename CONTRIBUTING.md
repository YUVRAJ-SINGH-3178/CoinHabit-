# Contributing

Thanks for contributing to CoinHabit. Please follow these guidelines to keep the project consistent and easy to maintain.

## Development Setup

1. Install Flutter SDK
2. Run `flutter pub get`
3. Configure `.env` using `.env.example`

## Branching

- Use short, descriptive branch names (e.g., `feature/goal-ui`)
- Keep PRs focused and small

## Code Style

- Follow `analysis_options.yaml`
- Prefer composition over large widgets
- Add comments only when logic is non-obvious

## Testing

- Run `flutter analyze`
- Add or update tests when you change logic

## Pull Requests

- Describe the change clearly
- Add screenshots for UI changes
- Mention any breaking changes
